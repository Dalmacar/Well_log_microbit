# Well Log Micro:bit Project

This project demonstrates how to simulate and visualize a **well log (Gamma Ray)** using BBC micro:bit devices and Python.

## 🔧 How to Use the Micro:bit Code

You can run the micro:bit programs in **two different ways**:

### 1️⃣ Using MakeCode (.hex file)

- Open the Micro:bit MakeCode website:  
  https://makecode.microbit.org/

- Upload the `.hex` file provided in this repository

- Flash it directly to your micro:bit

👉 This is the easiest option for beginners

---

### 2️⃣ Using Python (copy & paste)

- Go to the Micro:bit Python editor:  
  https://python.microbit.org/

- Copy and paste the `.py` code from this repository

- Download and flash it to your micro:bit

👉 This option is recommended if you want to understand or modify the code

---

## 📡 System Overview

- One micro:bit acts as a **controller (sender)**
- One micro:bit acts as a **receiver (car)**
- Data is sent via radio
- The receiver sends data to the computer via USB
- A Python script plots(microbit_gr_log) the data in real time
